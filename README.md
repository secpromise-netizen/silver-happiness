# OrangeFox Recovery for Troika

OrangeFox Recovery R11.3 (Unofficial) build for Troika device.

## Downloads

- **Recovery Image**: `fox_12.1/out/target/product/troika/OrangeFox-R11.3-Unofficial-troika.img`
- **Recovery ZIP**: `fox_12.1/out/target/product/troika/OrangeFox-R11.3-Unofficial-troika.zip`

## Installation

Flash the recovery image using fastboot:
```bash
fastboot flash recovery OrangeFox-R11.3-Unofficial-troika.img
```

Or install via existing recovery using the ZIP file.

## Build Info

- Version: R11.3
- Device: Troika
- Android Version: 12.1
- Build Type: Unofficial
